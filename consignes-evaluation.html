Evaluation du site web

Vous devrez vous présenter le site web que vous avez crée + le dépliant (env. 7 min par personne, en indiquant de quoi s'agit votre site, les pages qu'elle contient, les changements CSS que vous avez fait et le contenu du dépliant). Vous allez :

  • Présenter le site web fini

  • Montrer sur Weebly comment vous avez édite le site

Le jour de votre présentation, vous devrez rendre aussi :

  • Un document d'une dizaine de pages expliquant la conception de votre site Web grâce en particulier à des captures d'écran lisibles type avant/après ainsi que la conception de votre dépliant (objectifs des supports, conception pratique, difficultés, …). (Cf. "Cours 4")

  • Le dépliant sur papier qui permettra de présenter l'enterprise montrée dans votre site web (Cours 3, Prochain TD)

  • Le dépliant aux formats odt et pdf Le dépôt de documents sera fait aussi sur Chamilo le jour que vous avez présenté votre site ("TC CI2 - Technologie de l'Information et de la Communication 3" > "Travaux" > Séléctionner le groupe et la section à laquelle vous appartient), et avant minuit, (example le groupe TC21A doit le rendre le mercredi 29/11, le TC21B le mardi 28/11, etc.)

Le site Web

Vous devez réaliser INDIVIDUELLEMENT un site Web sur la plateforme Weebly concernant une entreprise fictive. L'enterprise peut comprendre :

  • Vente des articles
  • Offre de services
  • Associations...

Pour votre site, il faudra : écrire le texte en parlant de l'enterprise, les services/produits proposés, séléctionner les images (pour chercher des images libre de droits, regarder ici, si aucune image vous convient, utiliser Google Image, en indiquant toujours à qui appartient la photo ou le site d'où elle a été prise. Dans le document explicatif, vous ajouterez une feulle annexe avec les url d'ou vous avez pris les photos), organiser le site (pages, sous-pages..), ajouter des formulaires de contact si besoin, etc. Bref, il s'agit de dévélopper la totalité du contenu du site. Ainsi, il est nécessaire de :

  • Séléctionner un modèle parmi ceux proposés par Weebly
  • Adapter et remplir ce modèle selon les besoins de votre enterprise

Début de la création du site web

Pour commencer, il faut aller sur Weebly et séléctionner sur la page "Souhaitez-vous vendre des produits en ligne?" l'option "Pas maintenant" (Si vous voulez mettre une boutique pour vendre d'articles, une boutique peut être ajoutée plus tard)

Edition du site web

Une des consignes principales (et considérées pour l'évaluation) est l'adaptation du modèle séléctionné. Pour ce faire, il y a deux types d'édition à utiliser :

  • Utiliser les options proposées par Weebly ("Thème" > "options du police/options du thème", etc )
  • Adapter la source CSS. Cette dernière devient très utile lorsqu'il y a des petits changements que le format du thème ne prend pas en compte

Edition du CSS

Pour modifier le CSS sur Weebly:

  • Aller sur "Theme" et cliquer sur "Edition HTML/CSS"

Pour savoir ce qu'on veut modifier dans votre site:

  • Aller sur le site que vous êtes en train de créer (Cliquer sur "Publier" pour obtenir le lien)
  • Cliquer avec le bouton droit de la souris et séléctionner l'option "Examiner l'élément"
  • Survoler avec la souris la section "Inspecteur", trouver la partie de la page web à modifier et regarder au côté droit le CSS correspondant "Règles"
  • Identifier le nom du fichier où le code CSS de la section à modifier. Il se trouve à côté droit de la même section)
  • Retourner sur l'édition de votre site
  • Aller sur "Thème"> "Editer HTML/CSS"
  • Trouver le nom du fichier identifié sur "Règles"
  • Trouver la section du code à modifier
  • Modifier le code et publier (sauvegarde de changements)

OBLIGATOIRE : Un minimum de 8 changements du type CSS à faire dans votre site, au plus de ceux faits avec les options données par Weebly

Attention I : Pensez à prendre de notes et des captures d'écran lorsque vous faites le site. Cela vous aidera à écrire le document qui expliquera la conception (Cf. "Evaluation du site web")

Attention II : Nous vous rappelons que les lois françaises s'appliquent et que vous seul êtes responsable du contenu de votre site.
